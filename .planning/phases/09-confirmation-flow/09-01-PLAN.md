**ConfirmCard component internals:**

State machine with `useState<'idle' | 'loading' | 'confirmed' | 'cancelled' | 'expired' | 'error'>('idle')`:
- `secondsLeft` initialized to `data.expiresInMinutes * 60`
- `txHash` string (set on successful confirm)
- `error` string (set on failed confirm)

Countdown timer via `useEffect`:
```typescript
useEffect(() => {
  if (state !== 'idle') return;
  const endTime = Date.now() + data.expiresInMinutes * 60 * 1000;
  const tick = () => {
    const remaining = Math.max(0, Math.floor((endTime - Date.now()) / 1000));
    setSecondsLeft(remaining);
    if (remaining === 0) setState('expired');
  };
  tick();
  const id = setInterval(tick, 1000);
  return () => clearInterval(id);
}, [state, data.expiresInMinutes]);
```

Confirm handler — `handleConfirm`:
- Set state to `'loading'`
- `fetch(\`${process.env.NEXT_PUBLIC_API_URL ?? ''}/confirm\`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ txId: data.txId, userId }) })`
- On `res.ok`: `setState('confirmed')`, `setTxHash(json.txHash ?? '')`
- On `res.status === 409`: `setState('confirmed')`, `setTxHash(json.txHash ?? '')` (already confirmed = success)
- On `res.status === 410`: `setState('expired')`
- On other error: `setError(json.message ?? json.error ?? 'Transfer failed')`, `setState('error')`
- On catch: `setError('Network error - please try again')`, `setState('error')`

Cancel handler — `handleCancel` (per D-08, local-only):
- `setState('cancelled')` — no backend call
